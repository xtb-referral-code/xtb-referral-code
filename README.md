## How to Apply the Referral Code

1. Visit [xtb.com](https://xtb.com)
2. Click on the account creation option
3. Enter your email address
4. Select **"Add Code"**
5. Type `XTBWELCOME`
6. Save the code and complete the remaining registration steps

### Mobile Registration

1. Download the **xStation app** from Google Play or App Store
2. Select your country
3. Enter your email address

5. Input `XTBWELCOME`
6. Save and continue with registration
